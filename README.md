# Ai-tlas

## What?

Creation d'un petit assistant qui est capable d'
identifier si une phrase est soit positif, soit 
neutre, soit negatif.
L'assistant porte le nom de `Ai-tlas`.

## Steps

1. Train `Ai-tlas`

    - Gather data (ok)
        - positif
        - negatif
        - neutre

    - Sanitize data
        - dictionnaire
        - mots vides
        - TF formula
        - TF.IDF formula
        - TF.IDF of a whole document

2. Test `Ai-tlas`

3. Add UI

4. Finalize project
